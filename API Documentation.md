# Users

## List all users
```
GET /users
```

## Create a user
```
POST /user/:username/:password
```

## Change password of user
```
PUT /user/:username/:password
```

## Delete user
```
Delete /user/:username/:password
```

## Authenticate user
```
POST /user/:username/:password/signin
```
